# Villa SG Child Theme

Child theme for Twenty Twenty-Five built for Gutenberg-first brochure websites.

## Included

- Global tokens in theme.json (colors, typography, spacing)
- Front page template for quick launch
- Reusable block patterns for hero and intro sections
- Frontend and editor CSS for style parity

## Install

1. Copy this folder into wp-content/themes/
2. Ensure Twenty Twenty-Five is installed
3. Activate Villa SG Child from Appearance > Themes
4. Open Appearance > Editor and customize templates and patterns

## Notes

- Font families in theme.json are Lora and Figtree.
- If fonts are not available in your stack, install them with a plugin or local hosting.
- Edit texts and sections directly from Gutenberg.
